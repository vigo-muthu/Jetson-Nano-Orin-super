Hardware Requirements
* MyCobot robotic arm
* USB serial connection (/dev/ttyUSB0)
* Camera compatible with OpenCV/V4L2
* NVIDIA Jetson
  
Software Requirements
* Python 3.8+
* OpenCV
* ipywidgets
* pymycobot
* Jupyter Notebook / JupyterLab
  
Installation
pip install opencv-python ipywidgets pymycobot

Verify Python version:

python3 --version
